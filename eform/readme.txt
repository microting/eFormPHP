EFORM - PHP EXAMPLE CODE TO COMMUNICATE WITH MICROTING EFORM API
================================================================

Getting started

1. Contact Microting at +45 66 11 10 66 and get an account for https://my.microting.com (Developers backend)
2. Retrieve token, srv_name, site_id from above link
3. Add a unit to your account
4. Run the code and fill in "Config (step 1)" section with the information obtained from Developers backend:
5. Click on "Save as default" to store the inputed data.
6. Click on "Clear" to clear all output fields.
7. Click on "Generate Elements" to generete xml.
8. Click on "Send (step 3)"
9. Click the "Check status (step 5)" button and eventually click the "Fetch (step 7)" button
10. If everything went well, you can delete the list from Microting by clicking the "Delete from Microting (step 9)" button
11. You may use "Custom test" to run your customs tests.
